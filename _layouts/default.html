Conditionals

if statements 

              
    if (conditionalStatement){
      // Runs if true
    }

if else statements 

              
    if (conditionalStatement){
      // Runs if true
    } else {
      // Runs if false
    }

else if statements 

              
    if (conditionalStatement1){
      // Runs if true
    } else if (conditionalStatement2) {
      // Runs if true
    } else {
      // Runs if false
    }

Switch statemnts 

              
    switch (someVariable) {
      case caseOne :
        // Something happens
        break;
      case caseTwo :
        // Something happens
        break;
      case caseThree :
        // Something happens
        break;
      default:
        // Something happens
        break;
    }


Loops

For Loop 

              
    for (let i = 0; i < 5; i++){
      // Do something
    }

While Loop 

              
    while (testCondition) {
      // Do something
    }

Do While Loop 

              
    do {
      // Do something
    } while (testCondition);

Keywords usable in loops break, exits a loop continue, skips to the next iteration

JSON

JavaScript Object Notation

  • "JavaScript Object Notation"
  • All property names are surrounded by double quotes
  • Nothing that requires computation is allowed
  • No comments are allowed
  • JSON.stringify() takes JavaScript and turns it into JSON
  • JSON.parse() takes JSON and turns it into JavaScript
